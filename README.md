# game
Developed an interactive memory-based game using OpenGL and
C with animated card flipping, particle effects, hover animations,
and dynamic UI components.
Implemented gamification features including scoring, leaderboard
system, accuracy tracking, levels, visual feedback effects, and
reward animations.
Designed responsive game mechanics with sound integration,
collision-free interaction logic, and real-time user feedback.
Built immersive UI elements such as animated buttons, progress
tracking, celebration effects, and interactive gameplay screens.
